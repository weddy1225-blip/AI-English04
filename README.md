

# 🍳 我的創意早餐店 (Breakfast Adventure)

### —— 專為國小四年級設計的英語聽力與邏輯互動遊戲

這是一個基於網頁技術（HTML/CSS/JS）開發的教育互動遊戲。透過模擬早餐店點餐的情境，幫助國小學生在趣味遊戲中練習英語聽力、數量認知以及邏輯判斷。

## 📖 專案簡介
在「我的創意早餐店」中，學生將扮演店長，聽取客人的英語口說訂單並正確出餐。本專案特別針對 **教育部 108 國小英語課綱** 內容進行設計，涵蓋了 3-4 年級必備的基礎單字與請求句型。

## 🎮 遊戲特色
* **隨機題庫系統**：內建 30 道精選題目，每次開局隨機抽出 6 題（Level 1~3 各兩題）。
* **階梯式難度設計**：
    * **Level 1**：基礎食材與 1-10 數量感。
    * **Level 2**：加入對比形容詞 (Big/Small, Hot/Cold)。
    * **Level 3**：複合邏輯與介係詞應用 (With, But no...)。
* **即時語音回饋**：整合 Web Speech API，提供純正的英語口說練習環境。
* **視覺化經營報告**：結算時統計用時、正確率、跳過次數與提示使用量。

## 📊 評分與等級制度
遊戲採用動態積分演算法，鼓勵學生挑戰「不看文字提示」完成任務。

### 1. 積分公式
$$總分 = (答對 \times 100) - (答錯 \times 20) - (提示 \times 10) - (跳過 \times 50)$$

### 2. 店長等級判定
| 等級 | 分數區間 | 達成建議 |
| :--- | :--- | :--- |
| **🥇 金牌店長** | 550 分以上 | 需全對且完全不使用提示 |
| **🥈 銀牌店長** | 400 ~ 549 分 | 答對大部分題目，使用少量提示 |
| **🥉 銅牌店長** | 399 分以下 | 順利完成遊戲，過程中較常參考提示 |

## 🛠️ 技術架構
* **核心技術**：HTML5, CSS3, JavaScript (Vanilla JS)。
* **語音系統**：Web Speech API (`window.speechSynthesis`)。
* **排版設計**：響應式設計 (Responsive Design)，支援平板與手機遊玩。
* **部署支援**：可直接於 GitHub Pages 託管運行。

## 🚀 操作說明
1.  **聽取訂單**：點擊 **🔊 喇叭** 聽取英文要求。
2.  **準備食物**：點擊右側食材櫃按鈕，食物會進入 **🛒 出餐盤**。
3.  **修改內容**：若點錯數量，可點擊餐盤中食材旁的 **✕** 移除。
4.  **求助提示**：點擊 **💡 提示**，第一次顯示英文原文，第二次顯示中文翻譯。
5.  **完成上菜**：點擊 **🔔 上菜** 進行比對。
6.  **查看報告**：完成 6 題後，系統會自動產出經營成果報告。

## 📝 課綱對應 (國小四年級)
* **字彙範圍**：Egg, Bread, Milk, Juice, Sandwich, Apple, Big, Small, Hot, Cold.
* **常用句型**：`I want...`, `Can I have...`, `...please.`
* **學習目標**：能聽辨日常生活用語並做出正確反應。
